# blog-fancy 前台原型设计

## 视觉风格：赛博暗夜

### 配色方案
```
主背景：#0a0a0f (深邃黑紫)
次背景：#12121a (卡片背景)
主色：  #6366f1 (靛蓝紫，primary)
强调色：#22d3ee (青色，accent/hover)
渐变：  linear-gradient(135deg, #6366f1 0%, #8b5cf6 50%, #22d3ee 100%)
边框：  rgba(99, 102, 241, 0.3)
文字：  #f1f5f9 (主文字), #94a3b8 (次文字)
```

### 动效哲学
- **进入动画**：元素从下方淡入上浮，错开 80ms
- **悬停**：卡片微上浮 + 发光边框 + 阴影扩散
- **页面切换**：共享元素过渡 + 模糊切换
- **滚动**：视差效果，文章卡片渐次浮现
- **交互反馈**：点击涟漪 + 按钮脉冲 + 数字滚动

## 交互细节

### 卡片悬停动效
```
正常状态：
  - 背景: #12121a
  - 边框: transparent
  - 阴影: none

悬停状态（0.2s ease-out）：
  - 背景: #1a1a25
  - 边框: 1px solid rgba(99, 102, 241, 0.5)
  - 阴影: 0 0 30px rgba(99, 102, 241, 0.2)
  - transform: translateY(-4px)
  - 缩略图: scale(1.05)
```

### 按钮动效
```
悬停：
  - 背景渐变流动
  - scale(1.02)
  - 光晕扩散

点击：
  - scale(0.98) → 1
  - 涟漪扩散动画
```

### 数字滚动
```
点赞/阅读数：数字从0滚动到目标值，1.5s
配合：粒子消散效果
```

## 技术实现

```
packages/blog-fancy/
├── src/
│   ├── components/
│   │   ├── ui/           # shadcn 组件
│   │   ├── article/      # 文章卡片、详情
│   │   ├── social/       # 关注、评论、私信
│   │   └── layout/       # 导航、侧边栏
│   ├── hooks/            # 自定义 hooks
│   ├── lib/              # 工具函数
│   ├── pages/            # 页面组件
│   └── styles/           # 全局样式
```

**依赖**：
- shadcn/ui + tailwindcss
- framer-motion
- react-markdown + rehype-highlight
- @tanstack/react-query
- zustand
- react-router-dom

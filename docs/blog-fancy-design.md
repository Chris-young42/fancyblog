# blog-fancy 前台功能设计

## 技术栈
- 状态管理：Zustand
- 路由：React Router
- 样式：Tailwind CSS
- HTTP：Axios + React Query
- Markdown：react-markdown + rehype-highlight

---

## 核心模块

### 1. 首页
- 文章列表（分页/无限滚动）
- 分类/标签筛选
- 热门文章侧边栏
- 推荐文章（基于浏览历史）

### 2. 文章详情
- Markdown 渲染 + 代码高亮
- 文章目录导航
- 点赞、收藏、分享
- 评论系统（楼层制）

### 3. 分类/标签页
- 按分类/标签聚合文章列表

### 4. 用户/作者页
- 作者信息 + 文章列表
- 关注/粉丝列表
- 个人资料编辑

---

## 社交功能

### 关注系统
- 关注/取关用户
- 粉丝列表展示
- 关注推荐

### 评论系统
- 评论楼层结构
- 回复通知
- @提及用户（自动补全）
- 评论点赞

### 私信系统
- 发送私信
- 消息列表
- 未读消息红点

---

## 个性化功能

- 暗黑/亮色主题切换
- 阅读进度条
- 阅读历史记录
- 收藏夹管理
- 响应式布局（移动端适配）

---
